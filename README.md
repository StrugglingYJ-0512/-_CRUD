# -_CRUD
생활코딩_web2_Node.js

생산성

단하나의 파일안에 html코드가 들어가 있고, 
이 코드만 바꾸면
1억 개의 웹 페이지의 내용을
한 번에 폭팔적이고 미스터라하게도 바꿀 수 있게 된다.

노드js기술로 그 순간순간에 웹페이지를 프로그래밍 적으로 
생성해 내기 때문에.

이전에는 사용자의 참여가 제한됨.

node.js로 만드는 웹 어플리케이션은 
사용자가 쓰기, 수정, 삭제를 웹을 통해서 할 수 있도록 제공 할 수 있다.

이로부터 웹은 폭팔적으로 커졌다.

*cf_ php, jsp, 장고 등은 node.js의 경쟁관계에 있는 기술들임.
